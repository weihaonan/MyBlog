如何在github上创建博客

摘要：

练习如何在github上创建博客

准备工作：

安装git和node.js

百度搜索git node.js直接下载安装即可

正文

：

申请github账户，创建github仓库



  • 登陆github，点New repository创建一个新的仓库，命名为：{username}.github.com（这里的{username}为你在github上的用户名,尽量保持一致，这样服务器才能找得到）

  • 进入这个名为{username}.github.com的仓库，点击右边的 Settings 进入仓库管理中心；选择options

  • 在options中找到GitHub Pages，里边列出了你发布的博客所在的域名，一般为http://{username}.github.io/,以后就可以通过这个域名访问你的博客


安装hexo，hexo是基于node.js


hexo是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

  • 随便创建一个文件夹，进入文件夹，右键选择Git Bash Here

  • 在弹出的命令提示符中输入$ npm install -g hexo-cli,全局安装hexo

  • $ npm install hexo –save，在本地安装hexo

  • 查看hexo是否安装正确，$ hexo -v

  • 初始化hexo $ hexo init，这时候可能会提示错误： please run hexo init on an empty folder and then copy your files into it；只需将你所在文件夹中的文件删掉，再重新hexo init

  • 安装生成器，$ npm install

  • 在本地运行hexo, $ hexo s -g

  • 打开浏览器，输入localhost:4000,就可以在本地看到你的个人博客了，按住Ctrl+C键即可停止


配置项



  • 打开根目录下的_config.yml

  • 设置deploy，repo设置为 https://github.com/{username}/{username}.github.io.git
